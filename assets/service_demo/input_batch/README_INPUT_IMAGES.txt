Place permitted demo road images here for local batch inference.

Example:
 assets/service_demo/input_batch/road_001.jpg
 assets/service_demo/input_batch/road_002.png

Then run from project root:
 run_batch_infer_service.bat

Results are saved under:
 seg/runs/capstone_batch_service/

Keep private road images and license-unclear images in the local environment.
For public GitHub, keep this folder empty except for this README and .gitkeep.

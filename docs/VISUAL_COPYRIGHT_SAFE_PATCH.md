# Visual Copyright Safe Patch

Figures in this release are created with basic shapes and original text. Third-party decorative icons and unverified materials are avoided so the public package has a clear rights boundary.

## Applied policy

- Architecture diagrams: custom boxes and arrows only.
- Project QA diagram: text-only QA badge.
- Repository map: file tree and simple cards.
- Evidence figures: tables, charts, and status blocks.

## Text policy

Visible labels are written in clear portfolio wording. Model names, metric keys, file paths, and output keys remain in English where that is the natural technical label.

# Localization Audit

この日本語版 release では、visible documentation と diagram labels を日本語に統一します。

## Audit targets

- `README.md`
- `index.html`
- `MODEL_CARD.md`
- `LICENSE`
- `NOTICE.txt`
- `docs/*.md`
- `docs/assets/*`
- `docs/github_assets/*`

## Release rule

- 旧版の language-specific files は公開版に残さない。
- 著作者表記は `Wonseok Kim` に統一する。
- 図版内 label は日本語にする。
- code identifiers、paths、metric names は必要に応じて英字のまま維持する。

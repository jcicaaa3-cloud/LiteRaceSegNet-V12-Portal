Model / baseline scope

LiteRaceSegNet と SegFormer-B3 の位置づけ

LiteRaceSegNet 本体、SegFormer-B3 baseline、public package の外に置く artifact を整理します。

提案モデル

LiteRaceSegNet は MobileNetV2/V3-style inverted residual の設計アイデアから着想を得た、直接実装の custom LiteIR-style lightweight CNN segmentation model です。

比較 baseline

SegFormer-B3 は comparison baseline/reference として別 path に置いています。LiteRaceSegNet の実装主経路は lite_race です。

公開 package の外

SegFormer-B3 pretrained weight、fine-tuned checkpoint、Hugging Face cache、private dataset、credentials は public package の外に置きます。script 実行後の artifact はライセンス確認後に扱います。

短い説明
LiteRaceSegNet is a custom LiteIR-style lightweight CNN segmentation model; SegFormer-B3 is used as a separate comparison baseline.
License scope

SegFormer-B3 baseline artifacts

SegFormer-B3 関連 code/script は optional comparison baseline 用です。weight や cache は public package の外に置きます。

関連 path

seg/transformer_b3/、train_segformer_b3.py、segformer_b3_adapter.py、02_SETUP_SEGFORMER_B3_HF.bat は SegFormer-B3 comparison baseline 用です。

SegFormer-B3 license note

SegFormer-B3 の参照元 NVLabs repo は research/evaluation reference として扱います。Hugging Face の nvidia/segformer-b3-finetuned-ade-512-512 は license 表示が other のため、artifact の扱いは別途確認対象にします。

setup script

02_SETUP_SEGFORMER_B3_HF.bat は Hugging Face SegFormer-B3 weight を local comparison 用に download/cache する script です。生成物はライセンス確認後に扱います。

MobileNet / TorchVision note
seg/core/network.py には torchvision mobilenet_v3_large を呼ぶ Mbv3EdgeNet class があります。V13 の説明主経路は lite_race です。TorchVision は BSD-3-Clause、Hugging Face Transformers は Apache 2.0 として扱います。

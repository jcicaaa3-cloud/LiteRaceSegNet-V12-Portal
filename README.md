# LiteRaceSegNet-V13 Research Portal

LiteRaceSegNet-V13 is a GitHub Pages-ready research portfolio built from the AWS experiment package generated for LiteRaceSegNet. It brings the model structure, ablation table, threshold setting, boundary-local metrics, visual sheets, and release scope into one static site.

## How to read the result package

The V13 package is centered on a recorded experiment protocol. The numeric tables describe the measured AWS validation run, while the 10-image visual subset is used to make the mask, overlay, and boundary behavior easier to inspect. Boundary-local recall is shown as a refinement target, so the result section reads like a research record rather than a marketing scorecard.

## Site structure

- `index.html` — V13 overview and reading route.

## Public release scope

Included: source code, static pages, public diagrams, summary tables, selected result visualizations, evidence CSV/Markdown files.

Separated from the public zip: raw private datasets, checkpoints, pretrained weights, API keys, `.pem` files, `.env`, cloud credentials, and personal environment files.

## Model and baseline layout

LiteRaceSegNet-V13 uses the custom LiteIR-style lightweight CNN path implemented in `seg/core/lightweight_race.py`. Its LiteIRBlock follows MobileNetV2/V3-style inverted-residual efficiency ideas, then combines detail/context branches, LiteASPP context, boundary-guided fusion, and auxiliary outputs for road-damage segmentation.

SegFormer-B3 appears in this repository as the Transformer comparison baseline. Its scripts live in `seg/transformer_b3/` together with `02_SETUP_SEGFORMER_B3_HF.bat`; pretrained weights, fine-tuned checkpoints, Hugging Face cache files, private datasets, and credentials are kept outside the public zip.

Short description:

> LiteRaceSegNet is a custom LiteIR-style lightweight CNN segmentation model. SegFormer-B3 is the separate Transformer baseline used for comparison.
